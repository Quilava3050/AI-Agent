⚠️ TROUBLESHOOTING:

❌ Maven not found?
   → Download: https://maven.apache.org/
   → Add to PATH environment variable

❌ Java not found?
   → Download: https://www.oracle.com/java/
   → Need Java 17 or newer

❌ npm not found?
   → Download: https://nodejs.org
   → Install Node.js (includes npm)

❌ Port already in use?
   → Check: netstat -ano | findstr :8080
   → Or change port in application.properties

See README.md or QUICK_START.md for more help!
